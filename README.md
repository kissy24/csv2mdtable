# csv2mdtable

ディレクトリ内の CSV ファイルを検索し、Markdown テーブル形式のファイル（`.md`）に一括変換する Python スクリプトです。

## 特徴

- ディレクトリ内の全ての `.csv` ファイルを自動的に検出します。
- CSV のエンコーディング（UTF-8 / CP932）を自動判別して読み込みます。
- 空のセル (`NaN`) を空文字として処理し、綺麗な Markdown テーブルを出力します。
- パッケージ管理に `uv` を使用しており、高速かつクリーンな環境で実行可能です。

## セットアップ

このプロジェクトでは [uv](https://github.com/astral-sh/uv) を使用します。

1. リポジトリをクローンまたは配置します。
2. 依存関係をインストールします（`uv` がインストールされている場合、実行時に自動で行われます）。

```bash
uv sync
```

## 使い方

変換したい `.csv` ファイルをプロジェクトのルートディレクトリに配置し、以下のコマンドを実行してください。

```bash
uv run main.py
```

実行後、CSV ファイルと同じ名前の Markdown ファイル（例: `data.csv` -> `data.md`）が生成されます。

## 依存ライブラリ

- `pandas`: データ操作
- `tabulate`: Markdown テーブル出力用
